# 🌐 Inline Styles, Basic Interaction & Server-Side Validation

## 📌 Overview

This project focuses on enhancing web forms with **inline styling, client-side interaction, and server-side validation**. It demonstrates how user input is validated both on the client and server, ensuring data accuracy and reliability.

---

## 🎯 Objective

To build a form-based application that validates user input using both client-side and server-side techniques and temporarily stores the validated data.

---

## 🏢 Project Source

This project was completed as part of a task provided by **Cognifyz**.

---

## 🛠️ Features Implemented

### ✅ 1. Enhanced HTML Forms

* Created more detailed and interactive forms
* Used inline styles for basic UI improvements

---

### ✅ 2. Client-Side Validation

* Implemented validation using inline JavaScript
* Checked inputs like:

  * Required fields
  * Email format
  * Basic input conditions
* Displayed instant feedback to users

---

### ✅ 3. Server-Side Validation

* Validated form data on the server using Node.js/Express
* Ensured secure and accurate data handling
* Prevented invalid data from being processed

---

### ✅ 4. Temporary Data Storage

* Stored validated data on the server temporarily
* Used in-memory storage (array/object)

---
